# 🎓 Windows Server 2012 - Hệ Thống Trắc Nghiệm

## 📚 Giới Thiệu

Website ôn tập trắc nghiệm Windows Server 2012 với 170 câu hỏi được tổ chức thành 17 chủ đề khác nhau. Website hỗ trợ cả chế độ luyện tập và thi thử với timer.

**🔗 Live Demo**: [Thêm link sau khi deploy]

## ✨ Tính Năng

### 🏋️ Chế Độ Luyện Tập
- Chọn từ 17 chủ đề khác nhau hoặc "Tất cả chủ đề"
- 10 câu hỏi mỗi lần luyện tập
- Xem đáp án và giải thích chi tiết
- Làm lại bất cứ lúc nào

### ⏱️ Chế Độ Thi Thử
- Tùy chọn số lượng câu hỏi (5-50)
- Tùy chỉnh thời gian làm bài (5-120 phút)
- Đồng hồ đếm ngược thời gian
- Question grid để theo dõi tiến độ
- Hiển thị thời gian làm bài trong kết quả

### 📊 Thống Kê
- Theo dõi lịch sử làm bài
- Điểm trung bình và điểm cao nhất
- Tỷ lệ đạt (≥80%)
- Lưu trữ 50 lần làm bài gần nhất

### 🎯 Các Chủ Đề

1. **Installation & Deployment** - Cài đặt và triển khai
2. **Active Directory** - Quản lý thư mục tập trung
3. **AD Objects** - User, Group, OU management
4. **PowerShell** - Tự động hóa quản trị
5. **Networking** - DHCP, DNS, IPv6
6. **Storage** - Disk management, RAID
7. **File Sharing** - NTFS, Share permissions
8. **Group Policy** - GPO configuration
9. **WDS** - Windows Deployment Services
10. **IIS** - Internet Information Services
11. **Certificates** - CA, SSL/TLS, PKI
12. **Advanced AD** - RODC, AD Recycle Bin
13. **VPN** - PPTP, L2TP, SSTP
14. **NPS** - Network Policy Server, RADIUS
15. **NAP** - Network Access Protection
16. **DFS** - Distributed File System
17. **FSRM** - File Server Resource Manager

## 🛠️ Deployment

Website là static HTML/CSS/JS, có thể deploy lên:

### GitHub Pages (Khuyên dùng)
1. Push code lên GitHub repository
2. Settings → Pages → Source: main branch
3. Nhận link: `https://username.github.io/repo-name/`

### Netlify
1. Drag & drop thư mục vào netlify.com
2. Nhận link ngay lập tức

### Vercel
1. Import từ GitHub
2. Auto deploy

Xem chi tiết trong file `DEPLOY-GUIDE.md`
